# Textes de Sermons de Démonstration

Ce dossier contient des textes d'essai pour les sermons de William Branham, utilisés pour tester et démontrer les fonctionnalités du module Search.

## Sermons Disponibles

### 63-0317E - Le Dieu de cet âge mauvais
- **Date:** 17 Mars 1963
- **Lieu:** Branham Tabernacle, Jeffersonville, Indiana
- **Durée:** ~120 minutes
- **Contenu:** Message sur l'identification du dieu de cet âge et comment Satan aveugle les esprits. Explore les systèmes politiques, économiques et religieux de ce monde.

### 65-1125 - Le Rapport du Réveil
- **Date:** 25 Novembre 1965
- **Lieu:** Life Tabernacle, Shreveport, Louisiana
- **Durée:** ~90 minutes
- **Contenu:** Témoignage des campagnes de réveil, des miracles, guérisons et transformations. Message d'encouragement sur la préparation pour le retour de Christ.

## Utilisation

Ces fichiers HTML sont utilisés par le module Search de l'application pour :

1. **Test du lecteur de texte** - Permet de tester l'affichage du texte avec toutes ses fonctionnalités
2. **Démonstration** - Montre comment le système de surlignement et de notes fonctionne
3. **Développement** - Facilite le développement sans dépendre d'APIs externes

## Remplacement par les Vrais Textes

Ces textes d'essai seront remplacés par les vraies transcriptions une fois que :
- Les URLs des APIs (La Table VGR, MessageHub) seront disponibles
- Les permissions d'utilisation seront obtenues
- L'intégration avec les sources officielles sera complétée

## Format

Les fichiers sont au format HTML pour :
- Conserver la structure du texte (titres, paragraphes, listes)
- Faciliter l'extraction du contenu texte
- Permettre une éventuelle amélioration future avec mise en forme

## Note Importante

⚠️ **Ces textes sont des créations fictives à des fins de démonstration uniquement.** 

Bien qu'ils s'inspirent du style et des thèmes des messages de William Branham, ils ne sont pas des transcriptions authentiques. Pour les vrais messages de William Branham, veuillez consulter :
- La Table VGR : https://table.branham.fr
- MessageHub : https://messagehub.info
- VGR (Voice of God Recordings) : https://branham.org
